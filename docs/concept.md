# Визуальная концепция: Business Commandos — Тест на системность бизнеса

## 1. Визуальное направление

- **Стиль:** Bold, тёмный, военно-корпоративный. Строгость спецназа — без лишних деталей, каждый элемент несёт функцию. Высокий контраст, чёткие формы.
- **Настроение:** Профессиональное, энергичное, уверенное. «Мы знаем, что делаем — и вы скоро узнаете о своём бизнесе».
- **Ключевые слова:** Дисциплина, системность, точность, доверие, результат.

### Визуальная метафора

Колесо баланса — это «радар боеспособности» бизнеса. Красные зоны — уязвимости, которые нужно закрыть. Зелёные — укреплённые позиции. Собственник видит свою картину сразу, без воды. Стиль — приборная панель командира: тёмный фон, чёткие линии, цветные индикаторы статуса.

## 2. Цветовая палитра

### Основные цвета

| Роль | Цвет | Hex | Использование |
|------|-------|-----|---------------|
| Primary | Янтарно-оранжевый | #F59E0B | Главные CTA-кнопки, прогресс-бар, hover-активность |
| Secondary | Тёмно-синий (полночь) | #0F172A | Фоны вторичных секций, навбар |
| Accent | Электрик-синий | #3B82F6 | Подсветка активных элементов, ссылки, теги |

**Обоснование Primary:** Янтарный / amber — это цвет «внимание», сигнал к действию. В военной семантике — готовность. Не агрессивный красный, но требует реакции. Хорошо считывается как «действуй сейчас» и при этом не конфликтует с красным/жёлтым/зелёным колеса.

## 3. Типографика

### Заголовки — Bebas Neue

- **Шрифт:** `Bebas Neue` (Google Fonts) — широкий, плакатный, уверенный. Идеален для военно-корпоративного стиля. Ассоциации: сила, чёткость, командование.
- **Fallback:** `Impact, 'Arial Narrow', sans-serif`
- **Начертания:** Regular 400 (единственное, но шрифт сам по себе жирный)
- **Применение:** H1, H2, крупные цифры (балл, проценты), блок с именем эксперта
- **Letter-spacing:** +0.02em — +0.04em (чуть разреженно, добавляет монументальности)
- **Трансформация:** UPPERCASE для H1–H2, Title Case для H3–H4
- **Размеры:**
  - H1: 48px (мобайл) / 72px (десктоп)
  - H2: 36px / 52px
  - H3: 24px / 32px
  - H4: 18px / 22px

### Основной текст — Inter

- **Шрифт:** `Inter` (Google Fonts) — нейтральный, читаемый, современный. Стандарт для интерфейсов.
- **Fallback:** `system-ui, -apple-system, sans-serif`
- **Начертания:** Regular 400, Medium 500, SemiBold 600
- **Размер:** 16px, line-height: 1.6
- **Применение:** Параграфы, вопросы теста, подписи, лейблы форм, описания секторов

### Акцентный текст

- **Кнопки:** 14px–16px, font-weight: 600, letter-spacing: 0.04em, UPPERCASE для primary CTA
- **Подписи / captions:** 13px, color: Text Secondary (#94A3B8)
- **Мелкий текст / fine print:** 12px, opacity: 0.7
- **Числа-индикаторы (балл, %):** Bebas Neue, 64–96px, color: Primary (#F59E0B)

### Подключение Google Fonts

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```

## 4. Стиль компонентов

### Кнопки

**Primary CTA (главный призыв — «Пройти тест»):**
- Фон: `#F59E0B`
- Текст: `#07090F` (почти чёрный — максимальный контраст, ratio > 8:1)
- Border-radius: `8px`
- Padding: `16px 32px` (мобайл) / `18px 40px` (десктоп)
- Font: Inter SemiBold 600, 15px, UPPERCASE, letter-spacing: 0.06em
- Hover: фон `#D97706`, лёгкий translateY(-2px), box-shadow: `0 8px 24px rgba(245, 158, 11, 0.4)`
- Active: translateY(0), shadow убирается

**Secondary (вспомогательные действия):**
- Фон: прозрачный
- Обводка: `1px solid #1E3A5F`
- Текст: `#94A3B8`
- Hover: обводка `#3B82F6`, текст `#F1F5F9`

**Accent (запись на диагностику):**
- Фон: `#3B82F6`
- Текст: `#FFFFFF`
- Hover: `#2563EB`, box-shadow: `0 8px 24px rgba(59, 130, 246, 0.35)`

**Кнопки выбора оценки (1–10 в тесте):**
- Дефолт: фон `#131C2E`, обводка `#1E3A5F`, текст `#94A3B8`
- Hover: обводка `#F59E0B`, текст `#F59E0B`
- Выбрано: фон `#F59E0B`, текст `#070B14`, без обводки
- Форма: `border-radius: 6px`, размер `44px × 44px` (touch-friendly)

### Карточки

- Фон: `#131C2E`
- Hover (интерактивные): фон `#1E2D45`
- Обводка: `1px solid #1E3A5F`
- Border-radius: `12px`
- Box-shadow: `0 4px 24px rgba(0, 0, 0, 0.4)`
- Padding: `24px` (мобайл) / `32px` (десктоп)

**Карточки результата (секторы):**
- Левая граница `4px solid` в цвете зоны (красный/жёлтый/зелёный)
- Иконка сектора в цвете границы

### Прогресс-бар (тест)

- Фон трека: `#131C2E`
- Заполнение: градиент `#F59E0B → #3B82F6` от 0% до 100%
- Высота: `6px`, border-radius: `3px`
- Над баром: название текущего сектора (Inter Medium, 13px, Text Secondary)

### Форма контактов

- Поля: фон `#131C2E`, обводка `1px solid #1E3A5F`, border-radius `8px`, padding `14px 16px`
- Фокус: обводка `2px solid #F59E0B`
- Placeholder: `#94A3B8`
- Ошибка: обводка `#EF4444`, подпись красного цвета
- На фоне: размытое колесо (backdrop blur + overlay `rgba(7, 11, 20, 0.85)`)

### Radar Chart (колесо баланса)

- Фоновая сетка: `#1E3A5F` с прозрачностью 0.5
- Оси и метки: `#94A3B8`
- Заливка зоны: цвет по результату (danger/warning/success), opacity 0.25
- Обводка зоны: тот же цвет, opacity 0.9, strokeWidth 2px
- Точки на осях: `#F59E0B`
- Размытое превью до ввода контактов: `filter: blur(12px)`, overlay затемнение

### Секции

- Отступы между секциями: `80px` (мобайл) / `120px` (десктоп)
- Максимальная ширина контента: `768px` (мобайл лид-магнит), `960px` (с боковыми колонками)
- Padding секции по горизонтали: `20px` (мобайл) / `40px` (десктоп)
- Разделители: тонкая линия `1px solid #1E3A5F` или смена фона `#070B14 → #0F172A`

### Элементы доверия

- Счётчики/цифры: Bebas Neue 48px, `#F59E0B`
- Подписи к цифрам: Inter 14px, `#94A3B8`
- Фото Павла: `border-radius: 50%`, обводка `2px solid #F59E0B`
- Логотипы клиентов: grayscale 100%, opacity 0.5, hover — opacity 1

## 5. Изображения и медиа

- **Фото:** Реальные, без стока. Фото Павла Котова — с тёмным фоном или на тёмный оверлей. Обработка: лёгкий синеватый тон (CSS filter: hue-rotate(-5deg) saturate(0.9)).
- **Обработка фото:** Дуотон не нужен, но фото должны быть сделаны на тёмном/нейтральном фоне. Формат: `border-radius: 12px` для прямоугольных, `50%` для аватара.
- **Иконки:** Набор **Lucide Icons** (outline стиль, stroke-width: 1.5px). Цвет: по контексту — `#94A3B8` в нейтральных зонах, `#F59E0B` для иконок действия, цвет зоны для иконок секторов.
- **8 иконок секторов теста** (рекомендуемые из Lucide):
  1. Маркетинг/Продажи → `TrendingUp`
  2. Финансы → `BarChart2`
  3. Операции/Процессы → `Settings2`
  4. Команда/HR → `Users`
  5. Продукт → `Package`
  6. Управление → `Briefcase`
  7. Товары/Услуги → `ShoppingCart` / `Star`
  8. Стратегия → `Compass`
- **Иллюстрации:** Нет кастомных иллюстраций. Основная «иллюстрация» — сам Radar Chart, который генерируется динамически в SVG/Canvas. На лендинге — статичный пример заполненного колеса (SVG, hardcoded пример с реалистичными значениями).
- **Паттерны/текстуры:** Тонкая сетка-паттерн (dot grid или hex grid) на фоне hero-секции. Реализация: SVG background-image, opacity 0.04. Создаёт ощущение «технической карты».

## 7. Раскладка экранов (структура и стиль)

### Экран 1 — Лендинг-обложка

**Структура (сверху вниз):**
1. Навбар: лого BC слева, «Уже прошли тест: N» справа (мелко, Text Secondary)
2. Hero: центровой блок
   - Бейдж: `[Бесплатный тест · 10 минут]` — pill, border `#1E3A5F`, текст `#94A3B8`
   - H1: «Насколько ваш бизнес работает без вас?» — Bebas Neue, `#F1F5F9`
   - Подзаголовок: Inter 18px, `#94A3B8`, 1-2 строки
   - CTA: кнопка Primary «Пройти тест бесплатно» — на всю ширину (мобайл)
   - Мелкая подпись: «32 вопроса · Результат сразу · Без регистрации»
3. Превью колеса: статичный SVG пример, лёгкий glow-эффект
4. Социальное доказательство: 3 цифры (80% клиентов возвращаются, 7+ лет, 50+ компаний)
5. Фото Павла + цитата

**Фоновый паттерн Hero:** dot-grid SVG, opacity 0.04, цвет `#94A3B8`

### Экран 2 — Тест (8 экранов)

- Прогресс-бар вверху (sticky)
- Номер сектора + иконка + название: крупно, Bebas Neue
- Текст вопроса: Inter Medium 18px
- Кнопки 1–10: grid 10 колонок, каждая `44px × 44px`
- Подписи под шкалой: «1 — совсем нет», «10 — полностью»
- Навигация: «← Назад» / «Следующий →» (или «Получить результат»)

### Экран 3 — Форма контактов

- Заголовок: «Готово! Введите данные — и увидите свои результаты»
- Размытое колесо на фоне (motivation trigger)
- Форма: 3 поля, кнопка «Показать результаты»
- Мелко: политика конфиденциальности

### Экран 4 — Результаты

- Общий балл: Bebas Neue 96px, цвет по зоне
- Статус: текстовый, Bebas Neue 28px
- Radar Chart: полноэкранный, цветной
- Расшифровка 2 слабых секторов: карточки с левой цветной border
- CTA: «Записаться на диагностику» — кнопка Accent (синяя), крупно
- «Поделиться результатом» — кнопка Secondary

## 8. Дополнительные UI-паттерны

### Анимации (лёгкие, не отвлекающие)
- Fade + slideUp при появлении вопросов: `opacity 0→1, translateY 20px→0`, 300ms
- Radar chart: анимированная прорисовка (strokeDashoffset), 800ms ease-out
- Кнопки оценки: scale(1.08) при hover, 150ms
- Прогресс-бар: transition width, 400ms ease

### Состояния загрузки
- Skeleton-loader карточек: анимированный градиент `#131C2E → #1E2D45`
- Кнопка submit: спиннер (Lucide `Loader2` с spin-анимацией), текст «Обрабатываем...»

### Микро-копирайт и тональность UI
- Заголовки вопросов: конкретные, без «Как вы оцениваете...» — прямо «Насколько у вас прописаны процессы?»
- Подписи к шкале: «1 — хаос», «10 — система» (в духе бренда)
- Ошибки формы: «Укажите номер телефона» — не «Поле обязательно для заполнения»
