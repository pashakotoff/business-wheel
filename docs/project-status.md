# Business Wheel — Статус проекта

**Последнее обновление:** 2026-03-02

## Что это

Интерактивный квиз-тест «Колесо бизнеса» для Business Commandos. Пользователь оценивает 8 секторов бизнеса, получает визуальную диаграмму (SVG-колесо) и рекомендации. Цель — лидогенерация на бесплатную диагностику с Павлом Котовым.

## Текущий статус: БЕТА — готов к финальному QA

Сайт работает, задеплоен на GitHub Pages:
`https://pashakotoff.github.io/business-wheel/`

## Что сделано

### Фазы 1–4: Полностью завершены
- Бриф, концепция, вайрфреймы, копирайт — всё в `docs/`
- Полный фронтенд: HTML/CSS/JS, адаптив, SVG-колесо

### Ключевой функционал
- **Квиз**: 40 вопросов (8 секторов × 5), шкала 1–5
- **SVG-колесо**: интерактивная диаграмма результатов с анимацией
- **PDF-чеклист**: скачиваемый PDF «10 признаков ручного управления» (jsPDF)
- **Персональная ссылка**: `#r=BASE64` в URL — восстановление результатов без повторного прохождения
- **CTA-кнопка в PDF**: золотая кнопка со ссылкой на результаты
- **Яндекс Метрика**: трекинг целей (start, complete, download_pdf, diagnostic_click)
- **Google Sheets**: отправка данных диагностики через webhook
- **Отзывы**: 3 отзыва на основе реальных кейсов BC (Максим, Роман, Александр)
- **Фото автора**: JPG-аватар Павла в обоих trust-блоках (лендинг + результаты)

### Стек
- Чистый HTML/CSS/JS, без фреймворков
- jsPDF для генерации PDF
- SVG для колеса результатов
- CSS custom properties, responsive design
