# Настройка Google Sheets для приёма лидов

## Шаг 1: Создание таблицы

1. Перейдите на [sheets.google.com](https://sheets.google.com)
2. Создайте новую таблицу
3. Назовите её: **BC — Лиды Колесо Баланса**

## Шаг 2: Добавление скрипта

1. В таблице: **Расширения → Apps Script**
2. Удалите всё содержимое из Code.gs
3. Скопируйте содержимое файла `src/google-apps-script.js` и вставьте
4. Нажмите **Ctrl+S** (сохранить)
5. Назовите проект: **Webhook Колесо Баланса**

## Шаг 3: Развёртывание

1. Нажмите **Развернуть → Новое развёртывание**
2. Нажмите шестерёнку → выберите **Веб-приложение**
3. Настройки:
   - Описание: `Приём лидов v1`
   - Выполнять от: **Я (ваш email)**
   - Доступ: **Все**
4. Нажмите **Развернуть**
5. Если попросит авторизацию — разрешите доступ к таблице
6. **Скопируйте URL** веб-приложения — он выглядит примерно так:
   ```
   https://script.google.com/macros/s/ABCDEF.../exec
   ```

## Шаг 4: Подключение к сайту

1. Откройте файл `src/js/main.js`
2. Найдите строку `WEBHOOK_URL`
3. Вставьте скопированный URL:
   ```js
   const WEBHOOK_URL = 'https://script.google.com/macros/s/ВАШ_ID/exec';
   ```

## Шаг 5: Проверка

1. Откройте URL скрипта в браузере — должен вернуть:
   ```json
   {"status":"ok","message":"Webhook Business Commandos активен"}
   ```
2. Пройдите тест на сайте и заполните форму
3. Проверьте, что лид появился в таблице на листе "Лиды"

## Обновление скрипта

Если нужно изменить скрипт после развёртывания:
1. Внесите изменения в Apps Script
2. **Развернуть → Управление развёртываниями**
3. Нажмите карандаш (редактировать)
4. Версия: **Новая версия**
5. Нажмите **Развернуть**

URL остаётся прежним, обновлять на сайте ничего не нужно.
