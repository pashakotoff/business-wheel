# Чек-лист перед запуском на продакшен

Все места, которые нужно обновить при переезде на рабочий домен.

## 1. URL сайта

| Файл | Строка | Что менять |
|------|--------|------------|
| `src/js/main.js` | `const SITE_URL = '...'` | Заменить `https://pashakotoff.github.io/business-wheel/` на продакшен-домен |
| `src/js/main.js` (PDF) | `doc.text('businesscommandos.ru', ...)` | Проверить, что домен актуален |

## 2. Яндекс Метрика

| Файл | Строка | Что менять |
|------|--------|------------|
| `src/js/main.js` | `const YM_ID = 105930063` | Проверить/обновить ID счётчика |
| `src/index.html` | `ym(105930063, "init", ...)` | Тот же ID — должен совпадать с main.js |
| `src/index.html` | `<img src="https://mc.yandex.ru/watch/105930063" ...>` | Тот же ID |

## 3. Google Sheets Webhook

| Файл | Строка | Что менять |
|------|--------|------------|
| `src/js/main.js` | `const WEBHOOK_URL = '...'` | Проверить, что Google Apps Script задеплоен и URL актуален |

## 5. Деплой

| Что | Действие |
|-----|----------|
| GitHub Pages | Сейчас деплоится из `src/` через GitHub Actions. При переезде на свой хостинг — удалить `.github/workflows/deploy.yml` или перенастроить |
| HTTPS | Убедиться, что на продакшен-домене настроен SSL |
| favicon | Проверить наличие favicon.ico |

## 6. SEO и мета-теги

| Файл | Что проверить |
|------|---------------|
| `src/index.html` | `<title>`, `<meta name="description">`, Open Graph теги — убедиться, что URL и описания актуальны |
